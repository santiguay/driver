                       サポートするデバイス、動作確認ファームウェアバージョン

                                                       セイコーエプソン株式会社

    本リリースでは、下記のデバイスをサポートします。動作確認ファームウェアバー
  ジョンを記載している機種については、記載されているバージョンのファームウェア
  を使用していただくことを推奨します。
  このバージョン以降で動作させて下さい。バージョン以前のものでも動作しますが、
  問題があった場合は、このバージョン以降で動作させてください。
    なお、以下の記載の中で、I/F欄に使用している記号は、S=Serial、P=Parallel、
  U=USB、E=Ethernet（無線も含む）、B=Bluetoothを示します。
    
    注：
    ・ [1]が付加されているデバイスのサポートOSは、以下のものに限定されます。
          Windows XP, Windows Embedded for Point Of Service,
          Windows Embedded Standard, Windows Embedded POSReady 2009  
    ・ [2]が付加されているデバイスのEthernet接続は、有線のみのサポートです。
    ・ POSプリンタのANKモデル、MICR、およびCheckScannerの動作確認は英語版のOSで
       のみ行っています。
